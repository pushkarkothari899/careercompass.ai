# 🧭 CareerCompass AI

> *The project that asked the wrong question — and found the right one.*

CareerCompass AI started as an AI-powered resume scorer and job recommendation platform. Think Internshala with an AI layer. It worked — but it wasn't solving the real problem.

---

## 💡 The Pivot

While building CareerCompass, we kept running into the same realization:

> Students don't struggle to **find** jobs. They struggle to understand **why they aren't getting shortlisted** — and what they need to fix.

Platforms like LinkedIn, Internshala, and Naukri already show you opportunities. None of them clearly tell you:

- How ready you actually are for a role
- What skills you're missing
- Why your profile isn't converting to interviews
- What you should learn next

That insight changed everything. CareerCompass evolved from *"find jobs"* to *"understand your gap"* — and eventually into a focused new project.

**→ This project evolved into [Ascent](#) — a skill-gap analysis platform built on an NLP + embeddings pipeline.** *(link coming soon)*

---

## ⚙️ What CareerCompass Does

- 📄 Resume upload with AI-powered strength analysis
- 👤 Career profile collection (goal, interests, education)
- 📊 Resume score with personalized recommendations
- 🕓 Evaluation history tracking
- 🔐 JWT-based authentication

## 🚀 Run Locally

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm start
``
